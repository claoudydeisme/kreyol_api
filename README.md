# Haitian Creole ⇄ English Public Translation API

This project provides a **public, domain-aware translation API** for Haitian Creole (Kreyòl Ayisyen) and English.

Its mission is to deliver **accurate, standardized, culturally appropriate, and safety-focused translations** that are more reliable than general-purpose translation tools, especially for:

- Healthcare
- Education
- Government
- Public services

This API prioritizes:
**Clarity · Accuracy · Safety · Dignity**

---

## Key Features

- Haitian Creole ⇄ English translation
- Domain-locked terminology
- Standard Haitian Creole orthography
- Healthcare-safe language enforcement
- Long-document translation with structure preservation
- Human-governed terminology approval
- Public, auditable datasets

---

## Intended Users

- Healthcare providers
- NGOs and humanitarian organizations
- Government agencies
- Educators
- Developers serving Haitian communities

---

## Non-Goals

- Creative rewriting
- Idiomatic embellishment
- Medical or legal advice
- Automatic synonym generation

---

## Status

This project is under active development and intended to be released as a **public linguistic infrastructure**.
