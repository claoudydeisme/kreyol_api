# Contributing Guidelines

Thank you for your interest in contributing to the Haitian Creole ⇄ English
Public Translation API.

This project is **public linguistic infrastructure**.
Contributions must prioritize **accuracy, safety, and dignity**.

---

## Who Can Contribute

We welcome contributions from:
- Linguists (especially native Haitian Creole speakers)
- Healthcare and public-service professionals
- Engineers
- Educators and domain experts

---

## Core Principles (Mandatory)

All contributions must respect:
- Standard Haitian Creole orthography
- No French or English grammatical leakage
- Semantic accuracy over literal translation
- Domain safety (especially healthcare and government)
- No hallucinations or added meaning

---

## Types of Contributions

### 1. Code
- Bug fixes
- Performance improvements
- Infrastructure enhancements
- Testing and validation

### 2. Terminology
- Propose new domain-specific terms
- Improve clarity and consistency
- Flag unsafe or ambiguous terms

### 3. Documentation
- Improve clarity of rules
- Add examples
- Translate documentation (with approval)

---

## Terminology Contributions

All terminology must go through the **proposal → review → approval** process.

Rules:
- One source term = one target term
- No synonyms in locked domains
- Conflicts result in rejection (HTTP 409)

---

## Code Standards

- Python 3.10+
- Clear, readable code
- No silent behavior changes
- Deterministic outputs only
- Unit tests for new logic

---

## Review Process

- All changes require review
- Linguistic changes require a qualified reviewer
- Healthcare-related changes require domain review
- Decisions are logged and auditable

---

## What We Do NOT Accept

- Creative rewriting
- Slang or non-standard Creole
- French-based constructions
- Medical, legal, or financial advice
- Changes that reduce clarity for non-technical users

---

## Code of Conduct

All contributors must:
- Be respectful
- Accept linguistic diversity
- Prioritize community safety
- Act in good faith

Language access is a matter of **equity and dignity**.
